Serviço de Notificações API

API RESTful para envio de notificações push através do Firebase Cloud Messaging (FCM)

v1.0.0 RESTful HATEOAS

Sobre esta API

Este serviço de notificações permite enviar mensagens push para dispositivos móveis através do Firebase Cloud Messaging (FCM). A API é RESTful e implementa o padrão HATEOAS para navegabilidade melhorada.

Ver Documentação Completa

Endpoints Principais

  • GET /api/notifications

    Lista todos os endpoints relacionados a notificações.

  • POST /api/notifications

    Envia uma notificação para um único dispositivo.

  • POST /api/notifications/bulk

    Envia notificações para múltiplos dispositivos de uma só vez.

  • GET /api/notifications/health

    Verifica se o serviço está operacional.

Como Usar

Esta API utiliza o formato JSON para comunicação e inclui links HATEOAS em todas as respostas para facilitar a navegação.

Exemplo de requisição para enviar uma notificação:

{
  "token": "dispositivo-fcm-token-123",
  "notification": {
    "title": "Nova mensagem",
    "body": "Você recebeu uma nova mensagem"
  },
  "data": {
    "type": "message",
    "messageId": "msg123"
  }
}
